Present Continuous

Se usa para acciones que se estan realizando en el mismo momento en el que se habla y para hablar del futuro.


Estructura:


Afirmativo

Sujeto + "to be" presente + verbo principal[ing]

Negativo

Sujeto + "to be" presente + not + verbo principal[ing]

Pregunta

Verbo to be + Sujeto + verbo principal[ing]?
